# RN_github

**使用react-native技术开发IOS、Android两个平台的应用**


### feature
- 可以订阅 50 多种编程语言;
- 可添加/删除编程语言，并支持自定义它们的排序;
- 可收藏喜欢的项目;
- 支持多种颜色主题自由切换;
- 自持自定义订阅关键字;
- 轻松将自己喜欢的项目分享给好友;

---

### detail

- reactc-native（0.59.9）
- redux
- react-navigation （3.X）
- GithubTrending （API）
